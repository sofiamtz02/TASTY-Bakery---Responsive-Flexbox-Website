# Lab 4 – Responsive Flexbox Website (Part 2)

## Overview

This project is a multi-page responsive bakery website created using HTML and CSS with Flexbox. The goal of the lab was to practice building a complete responsive layout, organizing page structure, and creating consistent navigation and styling across multiple pages.

The site represents a fictional bakery called **TASTY Bakery** and includes several pages that demonstrate responsive design patterns and form layout.

---

## Learning Objectives

* Apply responsive web design principles
* Use Flexbox for page layout and alignment
* Build a multi-page website with consistent navigation
* Create and style forms using HTML and CSS
* Organize CSS for reusable components and page-specific styles
